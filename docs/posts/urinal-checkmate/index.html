In the Summer of 2015, some new Frameworks were popping out, and I was spending some time off work trying to learn them. First, I tried was Angular, which unbeknownst to me was going to be the one I was going to use most. Then I tried React, which I considered a dumspter fire and discarded right away.

React is the most popular JavaScript framework in the universe today, and sucked all my joy out of Web development.

My favorite of those frameworks became Vue. I didn't have to "learn" it, it already worked the way I used vanilla JavaScript.

This is what I came up with playing with the first version of Vue. The rules of the game I learned from a joke somewhere, and I created a few puzzles based on it. Nothing fancy.

I consider it now my first videogame.

Please, enjoy it.
